# *AI-Bits es un nuevo lenguaje de programación versátil e inteligente en la era de la IA*

### ✨ &nbsp;¿Por qué se llama AI-Bits?

Se pensó en el nombre de AI-Bits debido a que se quiso combinar en un solo nombre la inteligencia artificial y los bits, como la unidad mínima de la información. Además, debido a que el lenguaje se complementa con funciones de inteligencia artificial pensamos que encaja muy bien con la palabra bits.

### ⚙️ &nbsp;Guia de usuario

🔔 Para utilizar AI-Bit en tu propio entorno se deben realizar los siguientes pasos:

<img alt="Night Coding" src="https://media4.giphy.com/media/3o6nUWbtquwtR05DAQ/200.webp?cid=ecf05e470zkai9cpg0z0zt8nkg0yfnqasc6wvod7so9tfd5w&ep=v1_gifs_search&rid=200.webp&ct=g" width="200" height="180" align="right"/>

- &nbsp;Realizar un gitclone al repositorio del proyecto (Necesita de todos sus archivos).
- &nbsp;En el archivo .env dentro de la ruta src\source colocar los valores correspondientes a AZURE_OPENAI_ENDPOINT y AZURE_OPENAI_API_KEY (se utilizan para las funcionalidades con IA).
- &nbsp;Instalar los siguientes paquetes: os, re, webbrowser, pandas, math, pyfiglet, openai, dotenv, speech_recognition y pyttsx3.        
- &nbsp;Asegurarse que los scripts Lexador.py, Parser.py, Interprete.py, apuntador_errores.py, chatbot_openai.py y dar_bienvenida.py se encuentren dentro de la ruta src\source.
- &nbsp;Asegurarse que shell.py se encuentre en la raiz, ejecutar shell. py que es el script de python que ejecuta el interprete y llama a las funcionalidades del lenguaje.
- &nbsp; ejecutar [instruccion] ejecuta las instrucciones en el shell y compilar "[archivo.bts]" --r permite leer las instrucciones desde un archivo (la extension siempre .bts).

⚠️**Nota-> Se necesita la palabra "ejecutar" siempre antes de cualquier instruccion, a menos que se use "compilar"**

### ***Operadores y tipos de datos***
🐶&nbsp;Operadores aritmeticos: mas(+), menos(-), por(*), entre(/), elevado(^)\
🦊&nbsp;Operadores de comparación: mayor(:), menor(<), mayor-igual(>=), menor-igual(<=), igual(=), ?(!=)\
🐱&nbsp;Operadores logicos: and, or, not, nand, nor, xor\
🐴&nbsp;keywords para estructuras condicionales: when, other-when, other-case, <> (reemplaza a {})\
🦄&nbsp;keywords para iteradores condicionales (while): wheel-while, do, <> (reemplaza a {})\
🦓&nbsp;keywords para iteradores automaticos (for): wheel, limit, freq, do, <> (reemplaza a {})\
🐮&nbsp;keywords para declaracion de funciones (def): task\
🐷&nbsp;Funciones especiales: factorial, particion, gradiente-comb\
🐭&nbsp;operadores especiales: navegar, traducir, preguntar-ai-experta\
🐹&nbsp;Tipos de datos: ENTERO, REAL, TEXTO

### ***Algoritmos***
&nbsp;Partición\
&nbsp;Gradiente combinatorio
### ***Sintaxis basica***
🍇&nbsp;operaciones básicas -> 3 mas 4, 7 elevado 2, 3 por 4, etc.\
🍈&nbsp;Declaración de variables -> box a: 0, box c: 7, box cd3: "Mi nombre", etc.\
🍉&nbsp;comparación -> a mayor b, a menor-igual b, 3 igual 3, a ? b, etc.\
🍊&nbsp;logicas -> (a mayor b) and (a ? b), not (a menor b) xor (a igual c).\
🍌&nbsp;decision -> when a mayor b do < box a: a mas 2 >.\
🥭&nbsp;ciclos -> wheel-while a mayor b do < box a:a menos 1 >, wheel i:0 limit 10 freq 2 do <box c:c mas i>, etc.\
🍓&nbsp;Funciones -> definir: task add(a,b) -> a mas b, llamar: add(2,3)\
🥝&nbsp;Otros algoritmos -> factorial n, a particion b, gradiente-comb 7\
🥑&nbsp;Funciones novedosas -> "Tema de interes" navegar "[opcion]", "Nombre" traducir "idioma"\
🥑&nbsp;Funciones novedosas -> "Nombre" preguntar-ai-experta "Nombre Chatbot"\
